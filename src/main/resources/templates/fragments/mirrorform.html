Subversion Repository (Source)

Specify the Subversion repository you want to mirror into a Git repository. If the Subversion repository has restricted access, you can define a username and password. For publicly accessible repositories just leave username and password empty.

Git Repository (Mirror)

The SVN repository is mirrored into a Git repository that is hosted by gitanizer. You can define the name of the Git repository under which it will be hosted as well as an optional username and password. If username and password are not specified, the Git repository will be publicly accessible.
